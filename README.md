# ChatGPT Dig Deeper

ChaptGPTに深堀り問答させるプログラムです。

## Usage

環境変数 `OPENAI_API_KEY` にAPI KEYを設定します

第1引数に問答させたい単語を入力すると、その単語について問答をはじめます。
ChaptGPTは答えがなくなると空文字を返すので、空文字がかえってくるまで問答させます。
